* Cilium's Beta Mutual Auth support is now deprecated and will be removed in a future version. (Backport PR cilium/cilium#47374, Upstream PR cilium/cilium#47162, @youngnick)

* Adds a new policy entity, `cluster-mesh`, which selects all endpoints in all meshed clusters. Also, makes the `cluster` and `cluster-mesh` create significantly fewer policy map entries. (cilium/cilium#46813, @squeed)
